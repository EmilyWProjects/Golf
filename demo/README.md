HOW TO RUN:

-Install docker
-Clone the repository from github 
-Build docker image with Dockerfile in repository
-Run the docker container


FEATURES:

List all members
List all tournaments
List all 

Members queries:
-find by name
-find by phone
-find by membership duration (months)
-find by tournament start date
-get all members

Tournaments queries:
-find by start date
-find by location
-get all tournaments
-get participants
